Kairos

Kairos es un programa realizado en Java para la creación y optimización de horarios escolares. Sus principales características son:

  • Permite introducir una serie de restricciones al modelo de horario buscado, de entre una lista de restricciones disponibles, fácilmente ampliable.
  • Utiliza un algoritmo genético para la búsqueda de horarios que cumplan con todas o la mayoría de las condiciones.
  • Los items del horario calculado pueden arrastrarse con el ratón y se recalcula automáticamente.
  • Resaltado de los items que incumplen alguna restricción
  • Corre sobre cualquier sistema operativo con máquina virtual Java instalada (testado sobre Windows y Linux).
  • Es de código abierto, con licencia GPL.
