A school timetable designer tool that provides both manual and automatic search for possible solutions.

Kairos is a Java program aimed to create and optimize school timetables. Its main features, among other, are:

  • Teachers, classrooms, courses, and restrictions are part of the model.
  • It uses a genetic algorithm (GA) to look for a possible solution of the model.
  • From a solution computed by the GA (which may be or not optimal), user can visually rearrange timetable items, recalculating automatically restrictions.
  • In the timetable window, items that don't verify a restriction are marked as red.
  • Currently, it implements the following restrictions:
    • Courses cannot have gaps between.
    • A teacher cannot/only can teach some days in some hours.
    • Teachers cannot exceed a maximum of hours each day
    • A teeacher cannot spread its courses in more than a give number of days
  • It is relatively easy to implement custom-made restrictions.
  • It should run without problems in any Java-capable OS (currently tested on Windows XP, Windows 7, Linux)
  • It is free software, licensed by the GNU Public License.

Limitations

Kairos is currently under development. Although a full usable version is avalaible, there are still some features missing:
  • Assignation of courses to clasrooms are static, that is, are given by the user at design time. So, it cannot be used for example when different groups have to share a common classroom as for example, a computer room.
  • Currently, Kairos doesn't support subgroups division such that a subgroup can be in a course while another subgroup is in another differente course.
  • Right now, the program is entirely in spanish, but I am currently doing the traslation to English.
